Introduction⌗

I am Flutter developer and I love vim.

Requirements⌗

  1. Vim or Neovim. I personally use Neovim because I just happen to like it.
  2. Vim-plug
  3. Dart and Flutter installation

Plugins⌗

  1. Colorscheme
  2. Treesitter for better syntax highlighting
  3. Dart Vim Plugin
  4. Coc Flutter
  5. Vimspector
  6. Vim Test
  7. Snippets(LSP)
  8. Completion(LSP)
